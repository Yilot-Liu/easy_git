# easygit

[![Download](https://img.shields.io/github/v/release/Yilot-Liu/easy_git?label=Download%20APK)](https://github.com/Yilot-Liu/easy_git/releases/latest)

一款基于 JGit 的 Android Git 客户端。

**Language**: Kotlin  **Min API**: 26  **Target API**: 36  **Git Engine**: JGit 5.13

## 功能

- GitHub OAuth 登录,自动获取仓库列表
- 克隆/拉取远程仓库,支持进度提示
- 文件改动查看、暂存/取消暂存、提交、推送
- 分支切换
- 网络状态检测与友好错误提示

## 致谢

- [JGit](https://www.eclipse.org/jgit/) — Git 操作引擎
